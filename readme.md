## 第二章 基础数据类型

### 2.1 整型

| 类型    | 占用字节 | 表示范围               |
|-------|------|--------------------|
| byte  | 1    | -128~127           |
| short | 2    | -2^15~2^15-1       |
| int   | 4    | -2^31~2^31-1(约21亿) |
| long  | 8    | -2^63~2^63-1       |

- long类型变量赋值需要在字面量后面加**l**或者**L**
- 一般开发中，使用int
